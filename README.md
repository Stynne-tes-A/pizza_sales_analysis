# 🍕 Pizza Sales Analysis | Data Analyst Project

## 🧠 Project Overview

This project focuses on analyzing pizza sales data to uncover meaningful business insights.
Using tools like **SQL, Python, Power BI, and Excel**, the goal was to transform raw CSV data into actionable insights and interactive dashboards.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify best-selling pizzas and categories
* Understand customer ordering patterns
* Track revenue trends over time
* Build dashboards for decision-making

---

## 🛠️ Tools & Technologies

* 🐍 Python (Pandas, NumPy)
* 🗄️ SQL (Microsoft SQL Server Management Studio)
* 📊 Power BI
* 📈 Microsoft Excel
* 📁 CSV Dataset

---

## 🔄 Workflow

### 1. Data Collection

* Imported raw pizza sales data from CSV files

### 2. Data Cleaning & Transformation

* Cleaned missing and inconsistent data using **Python (Pandas)**
* Structured data for analysis

### 3. Data Processing (SQL)

* Loaded data into **SQL Server Management Studio**
* Performed data querying and transformation
* Created views and aggregations for analysis

### 4. Data Analysis

* Used SQL to answer key business questions like:

  * Total revenue generated
  * Most popular pizza types
  * Peak order times
  * Sales by category and size

### 5. Data Visualization

* Built interactive dashboards using:

  * **Power BI** (main dashboard)
  * **Excel** (additional analysis & charts)

---

## 📊 Key Insights

* 🍕 Identified top-selling pizza categories
* ⏰ Peak sales occur during specific hours/days
* 💰 Revenue trends show seasonal patterns
* 📦 Certain pizza sizes contribute more to revenue

---

## 📈 Dashboard Highlights

* Sales overview (Revenue, Orders, Quantity)
* Top 5 pizzas by revenue and quantity
* Sales by category and size
* Time-based analysis (daily/hourly trends)
